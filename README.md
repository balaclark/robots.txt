## Robots.txt

Displays the robots.txt of the file you wish to /robots.txt.

This lends itself from the favicon middleware.

## Installation

    npm install --save robots.txt

## Usage

    var robots = require('robots.txt')
    
    app.use(robots(__dirname + '/robots.txt'))
